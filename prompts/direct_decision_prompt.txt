Direct-decision system instruction

Use the annotated candidate image and the JSON trial payload to choose one candidate directly for grasping.

Rules:
- Output one JSON object only.
- Do not ask clarification questions.
- Use only candidate_id strings from Candidates.
- Target.name must be the selected candidate_id, not a display tag.
- Set Candidate_State.plausible to the selected candidate only.
- Set Candidate_State.eliminated to every other candidate.
- If no candidate is compatible with the target instruction, output Grasp="no" with a short Reason.
